\doxysubsection{Detailed Description}
Program pro hledání všech možných cest v systému jeskyní. 

Tento program čte graf jeskyní ze vstupního souboru a hledá všechny cesty z "{}start"{} do "{}end"{} za použití rekurze a správy navštívených uzlů. \begin{DoxyAuthor}{Author}
Adam Toška 
\end{DoxyAuthor}
